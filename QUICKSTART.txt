ğŸ” TROUBLESHOOTING

Problem: Server won't start
Solution: 
  1. Check if Python is installed: python --version
  2. Install dependencies: pip install -r requirements.txt
  3. Check if port 5000 is free

Problem: No networks detected
Solution:
  1. Run as Administrator (Windows) or sudo (Linux/Mac)
  2. Check WiFi is enabled
  3. App will use mock data automatically

Problem: Dashboard not updating
Solution:
  1. Check Socket.IO connection (green dot in header)
  2. Open browser console (F12) for errors
  3. Click "Refresh" button
  4. Restart the server

Problem: Import errors
Solution:
  pip install --upgrade -r requirements.txt
